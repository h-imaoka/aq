# aq (Athena Query CLI)

Command Line Tool for AWS Athena (bq command like)

## Installation

Install by gem:

```bash
$ gem install aq
```

## Usage

### query

Run query

```bash
$ aq query 'SELECT * FROM "test"."test_logs" limit 10;' --bucket=aws-athena-query-results-xxxx-region
```
